# Virtual_Gaming_Startup_Case_Study

A Revenue data dump for a Gaming App for the Month of April is given that contains the Daily revenue collected from each user visiting the App on each day of April

Questions Answered:

1. The top-5 states in terms of Revenue offered in the Month of April. How much revenue was collected from each of these states
2. The average number of users active on the App per day.
3. The average revenue collected per user per day
4. Any pattern in how many users visit the App daily.
